OpenFX : Base API Documentation

Introduction

OFX documentation comes in three forms
  • Docbook based documentation, generated into two formats of HTML,
  • Doxygen html documentation generated from comments in header files,
  • Comments in the raw header files.
The source to the documentation, in the form of C header files, XML files, perl scripts and several other bits and pieces area available from the openfx website.

Programing Reference

The OFX Programming Reference is mostly complete, and replaces the origin ungainly doxygen based reference pages. This documents all the functions, structures, actions and properties in the API.

Programming Guide

The OFX Programming Guide is very incomplete, it is new and is a guide on how to write plugins for the API.

Doxygen Documentation

The doxygen documentation extracted from the header files is a useful adjunct to the programmming reference.
